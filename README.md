
# Flipkart Grid

Ecommerce Platform 



## Product Requirement

php >= 8.0

composer >=2.1

mysql 
## Deployment

to run this project use command
```bash
   type .env.example > .env
```
after this command change the following credential according to your requirement 

```bash

    DB_CONNECTION=mysql
    DB_HOST=
    DB_PORT=
    DB_DATABASE=
    DB_USERNAME=
    DB_PASSWORD=

```

```bash
  composer install
```

```bash
  php artisan serve
```
